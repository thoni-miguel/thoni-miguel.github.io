Resolvr

Turn chaotic thoughts into structured, actionable plans with the help of journaling and ChatGPT.
Beat the procrastination cycle!

Workflow:

  1. Capture Thoughts: Use new_journal.py to create a new journal entry and quickly log your current tasks, ideas, or goals.
  2. Process Journal: Run resolvr.py to organize your journal entry into a clear prompt, ready to paste into ChatGPT for structured planning.

Scraplet

A lightweight, developer-friendly CLI tool for extracting HTML content from any website using CSS selectors.
Supports static pages and JavaScript-rendered content!

Workflow:

  1. Choose Scraping Mode: Use either scraplet_static.py for fast, HTML-only pages or scraplet_js.py to extract from JavaScript-heavy sites.
  2. Run Scraplet: Provide the target --url and CSS --selector to extract the data you need.
  3. Print or Persist: Output is printed to terminal by default. JSON/CSV output support is planned.
